# Anirban Das Portfolio

Static portfolio site for Netlify.

## Deploy

Upload this folder to Netlify or connect it as a static site. There is no build command and no publish subdirectory; the root folder is the site.

## Files

- `index.html` - page content
- `styles.css` - responsive styling
- `script.js` - small scroll/reveal interactions
- `assets/hero-desk.png` - generated hero visual
- `assets/anirban-portrait.jpg` - personal portrait
- `assets/Anirban-Das-Resume.pdf` - resume download
